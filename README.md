# Magento QA Automation

## Features
- Signup Test
- Login Test
- Logout Test
- Change Password Test

## Tech Stack
- Selenium
- PyTest
- Python

## How to Run
```bash
pip install -r requirements.txt
pytest tests/
```

## Folder Structure
- `tests/`: Test cases
- `pages/`: Page Object Model
- `utils/`: Config
- `screenshots/`: Store execution proofs
